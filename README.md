# Java Masterclass App

A responsive Flutter application for a Java course website. 

## Features
- Modern, clean landing page showcasing the course curriculum
- Responsive Course Player:
  - Side-by-side video and curriculum view on Desktop/Web
  - Stacked scrollable view on Mobile
- Mock data system with Modules and Lessons

## Tech Stack
- Flutter
- Dart

## Setup
1. Ensure Flutter is installed.
2. Run `flutter pub get`
3. Run `flutter run` to launch on your preferred device.
